NBA Player Collection Manager

Folder Name:
nba-collection-manager

Place this folder inside:
xampp/htdocs/

Run XAMPP and start Apache.

Open in browser:
http://localhost/nba-collection-manager/

Description:
This is a local CRUD web application that manages an NBA player collection.
Data is stored in JavaScript and persisted using localStorage.
The application starts with 30 preloaded NBA players.

Features:
- Create, Read, Update, Delete players
- Input validation
- Delete confirmation dialog
- Stats view with total players, average PPG, and position counts
- Data persists after browser refresh
